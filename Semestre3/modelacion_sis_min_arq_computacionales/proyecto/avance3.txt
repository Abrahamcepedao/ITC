/ Abraham Cepeda - A00827666
/  Tomando como referencia un sensor que entrega una lectura a través de un bus paralelo a razón de 12 bits/ms, realizar un programa dentro del entorno de MARIE, que almacene las primeras 50 mediciones en memoria.

Load ten
Store count

storeLoop, Input
	StoreI apuntador
   	LoadI apuntador
    Store check
	Load check
    Subt end
    Skipcond 000
    Jump done
    Load apuntador
    Add one
    Store apuntador
    
    Load count
    Subt one
    Store count
    Skipcond 400
    Jump storeLoop
    Load ten
    Store count
    Jump subtLoop
    
subtLoop, Load apuntador
	Subt one
    Store apuntador
    Load count
    Subt one
    Store count
    Skipcond 400
    Jump subtLoop
    Load ten
    Store count
    Jump sumLoop
    
    
sumLoop, Load acum
		 AddI apuntador
         Store acum
         Load apuntador
         Add one
         Store apuntador
         Load count
		 Subt one
         Store count
         Skipcond 400
         Jump sumLoop
		 Load ten
	     Store count
         Jump aveLoop

aveLoop, Load acum
		 Subt ten
         Store acum
         Load prom
         Add one
         Store prom
         Load acum
         Skipcond 000 /acum menor a cero?
         Jump aveLoop
		 Load prom
         Subt one
         Store prom
         Output
         Load cero
         Store prom
         Store acum
         Load ten
         Store count
         Jump storeLoop

done, Halt



/ Apuntadores
apuntador, hex 050  / apuntador principal

/ Variables
count, dec 0 / cantidad de numeros
acum, dec 0 / acumulador para sacar promedio
prom, dec 0 / calcular promedio
check, dec 0 /  para revisar si se ingreso 3567
end, dec 3567 / numero para terminar programa

/ Numeros
cero, dec 0
one, dec 1
ten, dec 10
